# Finance-Dashboard-💵 

A responsive **Finance Dashboard** built with **HTML, CSS, and JavaScript**.  
It provides a clear overview of income, expenses, and savings, with interactive charts powered by **Chart.js**.


## 🚀 Features
- Sidebar navigation for quick access to sections
- Summary cards showing:
  - Total Income
  - Total Expenses
  - Savings
- Interactive doughnut chart for financial distribution
- Recent transactions table
- Responsive design for desktop and mobile
- Easy to extend with more charts or tables


## 🛠️ Tech Stack
- **HTML5** – structure
- **CSS3** – styling
- **JavaScript (ES6)** – interactivity
- **Chart.js** – data visualization
